# CART Decision Tree From Scratch

This project implements a CART-based decision tree classifier from scratch in Python using the Forest Cover Type dataset.

The implementation focuses on understanding the optimization process behind decision trees, including recursive tree construction, Gini impurity minimization, and feature-threshold split selection.

## Key Features
- CART decision tree implemented from scratch
- Recursive binary tree construction
- Gini impurity optimization
- Feature importance analysis
- Hyperparameter tuning using validation data
- Benchmark comparison with scikit-learn

## Results
- Achieved ~79.5% test accuracy on a 7-class classification problem
- Produced performance comparable to sklearn's DecisionTreeClassifier
- Explored overfitting, feature importance, and computational trade-offs

## Tools
Python | NumPy | Scikit-learn | Matplotlib
